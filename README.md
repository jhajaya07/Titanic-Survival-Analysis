# Titanic-Survival-Analysis

This project performs Exploratory Data Analysis (EDA) on the famous Titanic dataset to uncover survival patterns among passengers.

## 📌 Objectives
- Analyze survival trends based on gender, passenger class, and age
- Handle missing data and clean the dataset
- Visualize insights using charts and graphs

## 🔍 Key Insights
- Females had a significantly higher survival rate than males
- First-class passengers were more likely to survive
- Children had better survival chances compared to older passengers

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Visualizations
- Survival rate by gender (bar chart)
- Survival rate by passenger class (bar chart)
- Age distribution (histogram)

## 🚀 Conclusion
This project demonstrates how data analysis and visualization can help uncover meaningful insights from real-world datasets.
